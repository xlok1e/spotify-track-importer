Как получить Spotify Client ID

Займёт около 3 минут

⚠ Для работы необходим аккаунт Spotify Premium.
1
Откройте Spotify Dashboard
Перейдите на developer.spotify.com/dashboard и войдите в свой аккаунт Spotify.
2
Создайте приложение
Нажмите Create app. Заполните любые название и описание — содержание не важно.
3
Укажите Redirect URI
В поле Redirect URI вставьте адрес этой страницы:
4
Включите Web API
Поставьте галочку напротив Web API. Нажмите Save.
5
Добавьте себя как пользователя
Перейдите на вкладку User Management → введите имя и email от вашего Spotify-аккаунта и добавьте себя.
6
Скопируйте Client ID
Вернитесь на вкладку Settings и скопируйте значение поля Client ID. Вставьте его в поле на этой странице.
